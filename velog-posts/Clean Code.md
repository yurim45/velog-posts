<h2 id="들어가며">들어가며,</h2>
<p>올해의 목표중 하나! <code>매 분기마다 개발서적📚 완독하기!</code> 의 첫번째!</p>
<p>1분기 목표로 <code>Clean Code</code>를 선택한 것은,</p>
<ul>
<li>개발자라면 한 번쯤 읽어봐야 할 것 같아서?!</li>
<li>이미 구매해둔 책인데 이번에 읽어야겠다는 생각?!
으로 시작했다!</li>
</ul>
<p>읽으며 가장 아쉬웠던 것은,
예시 코드가 내가 모르는 자바를 사용하기에 코드로 이해시키려고 하는 저자의 의도를 다 파악하지 못한 것.. 
(_자바 배울떄 다시 한번 재독해야겠다._💪)</p>
<p>그럼에도, 나의 개발 습관을 바꾸기에 충분했다.</p>
<h3 id="💪💪-바꿔보자-지키자">💪💪 바꿔보자! 지키자!</h3>
<ul>
<li>평소 변수명, 함수명 등 이름 짓기를 너무 어려워 했는데 어려워도 정성을 다해 지어야 겠다는 마음가짐과</li>
<li>알고 있는 내용(함수는 한 가지 기능만 하게!!)을 지켜야 겠다는 다짐과</li>
<li>불필요한 주석을 없애고 코드로서 의도를 전달하게끔 하자라는 의지!</li>
<li>그리고, TDD의 중요성에 대해 다시 한 번 느끼게 되는 계기가 되었다</li>
</ul>
<p>이 책에 대해 완벽히 이해한 것은 아니지만(_그러기엔 나에겐 너무 어렵다_😧😵;)
나중에 좀 더 성장하고 시간이 좀 더 흐른 뒤에 꼬옥~!! 재독해야겠다!!!!🫠🫠</p>
